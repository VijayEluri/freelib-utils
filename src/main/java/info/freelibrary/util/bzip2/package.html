This BZip2 package comes from the Ant BZip2 package (which is based on the work of Keiron Liddle, Aftex Software). The BZip2 classes were extracted from the Ant project and made available as a separate package by Kohsuke Kawaguchi. The FreeLibrary project then took these separate BZip2 classes and put them in the info.freelibrary package. They are licensed under the APL 2.0 license.
